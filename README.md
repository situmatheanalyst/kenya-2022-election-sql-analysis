# kenya-2022-election-sql-analysis
# A Statistical Analysis of the 2022 Kenyan Presidential Election

## 🔷 Executive Summary
The 2022 Kenyan Presidential Election was highly competitive between **William Ruto** and **Raila Odinga**.

Out of **14,213,137 total votes cast**:
- William Ruto: **7,176,141 votes (50.49%)**
- Raila Odinga: **6,942,930 votes (48.85%)**

This reflects a **narrow victory margin (~233,211 votes)**, highlighting a **highly polarized electorate**.

---

## 🔷 Project Objective
To analyze voting patterns, turnout, and electoral dynamics across Kenya using structured data and SQL.

---

## 🔷 Data Description
The dataset includes:
- All **47 counties + diaspora votes**
- Registered voters
- Total valid votes
- Candidate vote counts
- Rejected ballots

---

## 🔷 Tools & Technologies
- MySQL
- SQL
- Data Cleaning & Validation

---

## 🔷 Methodology
The analysis involved:
- Vote aggregation using SQL queries
- Regional grouping of counties
- Voter turnout computation
- Margin analysis between candidates
- Identification of swing counties

---

## 🔷 National Results
- 🥇 **Winner:** William Ruto (50.49%)
- 🥈 Raila Odinga (48.85%)
- 📊 Margin: ~233,211 votes

---

## 🔷 Regional Voting Patterns
- **Raila Odinga strongholds:** Nyanza, Western  
- **William Ruto strongholds:** Rift Valley, Mount Kenya  

---

## 🔷 Voter Turnout Analysis
- Highest turnout:
  - Rift Valley: **71.05%**
  - Nyanza: **69.54%**

- Lower turnout:
  - Coast: **50.75%**
  - Nairobi: **55.42%**

---

## 🔷 County-Level Insights
- **Raila strongholds:** Kisumu, Homa Bay, Siaya  
- **Ruto strongholds:** Kiambu, Kericho, Bomet  

---

## 🔷 Swing Counties
- Lamu  
- Isiolo  
- Tana River  
- Kajiado  
- Narok  

---

## 🔷 Rejected Ballots
Rejected ballots were generally **below 1%**, indicating efficient electoral processes.
